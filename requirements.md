# Proyecto: SQL Server Sales Analytics
## 1. Contexto

Proyecto desarrollado como freelance para un cliente pequeño–mediano, con la necesidad de analizar ventas, clientes y productos directamente en SQL Server, sin capas adicionales de visualización o automatización.

## 2. Objetivo

Construir un modelo relacional y consultas analíticas en SQL Server (SSMS) que permitan responder preguntas clave de negocio de forma confiable y reproducible.

## 3. Alcance 

* Incluido
  * Diseño de modelo de datos relacional
  * Creación de tablas en SQL Server
  * Carga de datos simulados
  * Consultas analíticas en T-SQL
  * Documentación técnica básica

* Fuera de alcance
  * Visualización (Power BI, dashboards, reportes)
  * Procesos ETL
    * Extraer de fuentes de datos fuera de alcance o de proveedores de servicios
    * Tranformacion de datos.
    * Carga de datos de 3ros o fuera de alcance.
  * Streaming de datos
  * Seguridad avanzada
  * Infraestructura cloud o productiva

## 4. Requerimientos funcionales

* Ventas totales por periodo

* Productos con mayor ingreso

* Clientes con mayor gasto

* Ventas por país y categoría

* Ticket promedio por cliente

## 5. Requerimientos no funcionales

SQL Server 2019+

Código legible y documentado

Consultas optimizadas para análisis

Ejecutable en SSMS
